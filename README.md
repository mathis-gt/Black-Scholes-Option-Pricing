    "## Explanation\n",
    "\n",
    "### Reason for Black-Scholes pricing\n",
    "\n",
    "The Black-Scholes model is a mathematical model used to estimate the theoretical value of European-style options. It is widely used in financial markets because it provides a consistent method of valuing options based on measurable market factors.\n",
    "\n",
    "The model considers several key variables:\n",
    "\n",
    "Current stock price.\n",
    "Strike price.\n",
    "Time until expiration.\n",
    "Risk-free interest rate.\n",
    "Volatility of the underlying asset.\n",
    "\n",
    "This library uses these inputs to calculate both call and put option prices.\n",
    "\n",
    "### How its calculated\n",
    "\n",
    "The Black-Scholes model first calculates two intermediate values, d1 and d2, which are used to estimate the probability of the option finishing in-the-money.\n",
    "\n",
    "These values are then used with the cumulative distribution function of the standard normal distribution to calculate the final option price.\n",
    "\n",
    "The call option formula is:\n",
    "\n",
    "C = S*N(d1) - K*e^(-rT)*N(d2)\n",
    "\n",
    "The put option formula is:\n",
    "\n",
    "P = K*e^(-rT)*N(-d2) - S*N(-d1)\n",
    "\n",
    "Where:\n",
    "\n",
    "N(x) is the cumulative distribution function of the standard normal distribution.\n",
    "e represents Euler's number.\n",
    "The other variables represent the user-defined model inputs.\n",
    "\n",
    "### Limitations\n",
    "\n",
    "The Black-Scholes model has several assumptions which can reduce accuracy in real-world situations.\n",
    "\n",
    "These include:\n",
    "\n",
    "The model assumes volatility remains constant over the lifetime of the option.\n",
    "Stock prices are assumed to follow a continuous log-normal distribution.\n",
    "Transaction costs and taxes are ignored.\n",
    "The model assumes European-style options which can only be exercised at maturity.\n",
    "Dividends are not considered without additional adjustments.\n",
    "\n",
    "Due to these assumptions, the output should be considered a theoretical estimate rather than a guaranteed market price.\n",
    "\n",
